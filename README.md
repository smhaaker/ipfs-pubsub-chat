# ipfs-pubsub-chat

> 
> ipfs pubsub chat client experiment
>

## Build Setup

``` bash
# Install dependencies
npm install

# Compile
npm run compile

# Run local server
npm start

# Open browser to
localhost:12345/index.html
```

Each window has a unique Peer ID. 

``` bash
# Chat commands
/commands - gives list of commands in web chat
```
