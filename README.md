# ipfs-pubsub-chat

> 
> ipfs pubsub chat client experiment
>

## Build Setup

``` bash
# Install dependencies
npm install

# Compile
npm run compile

# Run local server
npm start

# Open browser to
localhost:12345/index.html
```

Each window has a unique Peer ID. 

chat commands:
/peers - gives a list of current peers


running in terminal:
node src/terminal.js 'peer to connect'
